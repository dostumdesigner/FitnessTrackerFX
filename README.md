### FitnessTrackerFX — трекер тренировок

### Описание проекта  
JavaFX-приложение для планирования и отслеживания тренировок.  
Позволяет создавать, редактировать, удалять и просматривать тренировки, а также смотреть простую статистику по активности.  
Подходит для людей, которые хотят вести личный дневник тренировок: кардио, силовые, растяжка и т.д.

### Основные функции:
• CRUD-операции с тренировками (создание, редактирование, удаление, просмотр)  
• Хранение данных о дате, типе тренировки, длительности, интенсивности и комментариях  
• Фильтрация тренировок по дате и типу (через поле поиска и выпадающий список)  
• Отдельное окно добавления/редактирования тренировки  
• Мини-статистика: количество тренировок, суммарная длительность, средняя длительность  
• При желании — отображение тренировок по неделям/месяцам (простое расписание)  
• Сохранение и загрузка данных в формате JSON  
• Валидация полей ввода и подтверждение удаления через диалоговые окна  
• Оформление интерфейса с помощью CSS (фон, цвета кнопок, шрифты, отступы)  
• Архитектура MVC и использование ObservableList для динамического обновления данных  

### Установка и запуск  
Убедитесь, что у вас установлен Java JDK 17 или выше.  
Скачайте или клонируйте репозиторий:  
```bash
https://github.com/dostumdesigner/FitnessTrackerFX.git
``` 

Откройте проект в IntelliJ IDEA или другой IDE с поддержкой JavaFX.  
Добавьте библиотеку JavaFX в зависимости проекта (через Gradle/Maven или настроив JavaFX SDK в настройках проекта).  
Убедитесь, что главный класс (например, App.java или Main.java) указан как точка входа.  
Запустите главный класс как Java Application.  
Приложение откроется, и вы сможете работать со своими тренировками.

### Примеры использования  
• Создание тренировки: нажмите кнопку «Добавить тренировку», заполните поля (дата, тип, длительность, интенсивность, комментарий) и нажмите «Сохранить» — тренировка появится в таблице.  
• Редактирование тренировки: выберите тренировку в списке и нажмите «Редактировать», измените поля и сохраните изменения.  
• Удаление тренировки: выберите тренировку в списке и нажмите «Удалить». Подтвердите действие во всплывающем диалоговом окне.  
• Фильтрация тренировок:  
  – выберите тип тренировки в выпадающем списке,  
  – введите текст в поле поиска или выберите диапазон дат — таблица автоматически отобразит только подходящие тренировки.  
• Просмотр статистики: перейдите на вкладку «Статистика» — будет показано общее количество тренировок, суммарное время и средняя длительность.

### Структура репозитория  
```bash
src/  
├─ model/           # Модель данных (классы Workout, WorkoutType, Statistics и т.п.)  
├─ view/            # FXML-файлы интерфейса и CSS-стили  
├─ controller/      # Контроллеры для управления логикой и обработкой событий  
└─ App.java / Main.java # Главный класс приложения (точка входа)

README.md           # Документация проекта  
testing/            # Документы по тестированию (план, отчёт, отзывы, issues)  
docs/               # Дополнительная документация (Agile, руководство пользователя и т.д.)  
```

### Технические требования  
• Язык: Java 17+  
• UI-фреймворк: JavaFX SDK (модули javafx.controls, javafx.fxml)  
• Архитектура: MVC (Model / View / Controller), использование ObservableList  
• IDE: IntelliJ IDEA, Eclipse или аналогичная  
• ОС: Windows / macOS / Linux  
• Дополнительно: библиотека Gson для работы с JSON-файлами
